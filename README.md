# UnsplashClient
画像のAPIを提供している[Unsplash](https://unsplash.com/ja)を利用してアプリの実装を行なっている。

## ページ構成
2画面の構成になっている
- 画像検索画面
- 画像詳細表示画面

## ポイント
- 画面遷移の実装
- Clean Architectureの実装
- Retrofit & Moshi ライブラリの使い方
